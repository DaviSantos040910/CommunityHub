O que este projeto demonstra

Funcionalidades Implementadas

Cada recurso abaixo foi construído do zero, integrando frontend e backend.

🔐

Autenticação e Sessão

Sistema completo de cadastro, login e logout com controle de sessão. Integração com login via Google (OAuth).

👥

Sistema Multiusuário

Cada usuário tem seu próprio perfil com avatar, bio e galeria de postagens. Dados isolados por conta.

❤️

Interações em Tempo Real

Sistema de curtidas assíncrono (AJAX) que atualiza sem recarregar a página, proporcionando fluidez na experiência.

🔍

Busca Inteligente

Motor de busca integrado com resultados instantâneos para postagens e usuários, sem recarregamento de página.

☁️

Upload em Nuvem

Imagens armazenadas no Cloudinary com CDN global. Persistência garantida independente do estado do servidor.

📱

Design Responsivo

Interface adaptável de desktop a mobile, com sidebar que se transforma em barra de navegação inferior.

Perguntas Frequentes

FAQ

Isso é um produto real?

O CommunityHub é uma demonstração de portfólio, desenvolvido para apresentar competências em desenvolvimento Full Stack: autenticação, sistema multiusuário, uploads em nuvem, interações assíncronas e deploy em produção. A temática anime é utilizada como exemplo de aplicação.

Dá para adaptar para o meu negócio?

Sim. A base de autenticação, perfis de usuário, upload de mídia e estrutura de comunidade pode ser adaptada para qualquer plataforma que precise de login, perfis e interações entre usuários.

Quais tecnologias foram usadas?

Python e Django no backend, PostgreSQL como banco de dados, JavaScript com AJAX no frontend, Cloudinary para armazenamento de mídia, e deploy automatizado no Render com Gunicorn.
